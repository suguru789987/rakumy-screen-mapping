# ラクミー経営管理画面 マッピング仕様書

ラクミー経営管理画面(会社画面・店舗画面)の指標マッピング仕様書です。

> **制作:** [@suguru789987](https://github.com/suguru789987) (PdM)
> **担当範囲:** 全画面の指標棚卸し / データソース特定 / 演算ロジックの明示 / バージョン v1 → v21 の継続更新
> **ツール:** Claude Code / TSV / Markdown
> **規模:** 会社画面151指標 + 店舗画面278指標 = **合計429指標**の完全マッピング

## ファイル構成

| ファイル名 | 説明 |
|-----------|------|
| `会社画面マッピングリスト_v21.tsv` | 会社画面の全指標マッピング（151指標） |
| `店舗画面マッピングリスト_v21.tsv` | 店舗画面の全指標マッピング（278指標） |

## マッピング内容

各指標について以下の情報を定義：
- 画面タイトル・指標名
- データソース（ZTOTAL, TTOTAL, インフォマート等）
- テーブル・カラム
- 演算ロジック
- インプット元・アウトプット先
